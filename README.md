# User FLow

## Instructions to run the app

### Clone the repository

git clone https://github.com/Shiraz-2022/userFlow.git

### Install depenencies

npm install

### Run the app

npm start

Live app at : - https://userflow2022.netlify.app/

### Tech stack used

React, Tailwind CSS
